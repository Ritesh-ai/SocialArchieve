Instructions

  • Click on the link provided.
  • Click on the Log In button if you are not logged in already.

  • Click on +Add a new app button on the right side.
  • Fill in all the required details and click the create app Id button.
  • Select Graph api Explorer options inside tools tab.
  • Click Get token drop down and select an option as per the requirement.
  • Select all the permissions from the pop up and click the Get Access Token button.
  • Copy the access token and paste it in the Access token field.
  • Click on Continue button.
